# Installing OpenTofu via MacPorts

You can use OpenTofu as a [standalone binary](standalone.mdx) or you can install it using [MacPorts](https://ports.macports.org/port/opentofu-1.9). OpenTofu is available in the MacPorts ports tree, so you can install it by running:

<CodeBlock language={"bash"}>{PortScript}</CodeBlock>

Validate the installation by running:

<CodeBlock language={"bash"}>{ValidateInstall}</CodeBlock>
